# 🤖 AI Deepfake Detection System

An AI-powered Deepfake Detection Web Application built using **TensorFlow, OpenCV, and Flask** that detects whether an uploaded human face image is **REAL or FAKE**.

---

## 🚀 Project Overview

Deepfake technology uses Artificial Intelligence to manipulate facial identities in videos and images. This project aims to detect such manipulated media using Deep Learning and Computer Vision techniques.

The system allows users to upload an image through a web interface and receive an AI prediction with confidence score.

---

## 📊 Dataset Used

This project uses the **DeepFake Videos Dataset** available on Kaggle.

🔗 Dataset Link:  
https://www.kaggle.com/datasets/unidpro/deepfake-videos-dataset

### Dataset Description
- Contains **10,000+ media files**
- Includes videos of **real and AI-generated faces**
- Features over **7,000 individuals**
- Designed for Deepfake Detection and Liveness Detection research
- Fake faces are generated and overlaid onto real videos for training detection models :contentReference[oaicite:1]{index=1}
